7. Error Analysis

Errors & Issues

ERROR 1: Left vs Right Door Ambiguity (CRITICAL for this test)

GroundingDINO detects "door" in 37 photos across nearly all zones. It CANNOT determine which doors are on the left vs right side of the building. The system detects all doors equally.

Impact: The user might be guided to the WRONG door (right side instead of left). The VLM must use spatial reasoning ("left wall") to distinguish.

Mitigation: The VLM prompt should emphasize: "Exit through the door on the LEFT wall. Do NOT go through doors on the right side." Photos with box coordinates showing doors on the left edge (low x-values) of the image can provide hints.

ERROR 2: Duplicate Label Format — "extinguisher fire extinguisher" (LOW)

The fire extinguisher label is detected as extinguisher fire extinguisher (redundant). This is a GroundingDINO text prompt artifact where the prompt "fire extinguisher" matches as a merged phrase.

Impact: Minor — the detection is still correct, just the label is verbose. No functional impact on navigation.

ERROR 3: Multiple Refrigerators Detected in Different Zones (MODERATE)

Refrigerator is detected in 18 photos across zones 1, 2, 5, 6, 9, and 10. Some of these may be the same refrigerator seen from different angles, but others could be different refrigerators (water coolers, mini-fridges).

Impact: The system might declare ARRIVED at a water cooler (misidentified as "refrigerator") instead of the actual refrigerator. IMG_1472 (score 0.85) shows the clearest, largest refrigerator and should be prioritized.

ERROR 4: Fire Extinguisher in Starting Zone (MINOR CONFUSION)

Fire extinguisher is detected at the START position (IMG_1434 at 0.69, IMG_1435 at 0.64). The system might try to declare "ARRIVED" for fire extinguisher before the user even moves.

Impact: If processing starts at Zone 2, the VLM might say "fire extinguisher already visible." The navigation goal requires finding it as a distinct stop, so the VLM should understand the sequential goal order.

ERROR 5: Cabinet Overdetection (LOW)

"Cabinet" is the most detected label (62 detections, 37 photos) at moderate confidence (avg 48.8%). Cabinets are everywhere in this office and don't help with navigation.

Impact: Wastes detection slots. Similar to "bathroom restroom" in the grocery test, but less severe because cabinet detections are generally correct (just not useful for navigation).

ERROR 6: VLM Processing Latency (OPERATIONAL)

First photo: ~30-60s warm-up. Subsequent: ~10-15s each. For Test 1 (56 photos): 60 + (55 x 12) = ~12 minutes total. For Test 2 (7 photos): ~3 minutes total.

Impact: Acceptable for office navigation. Much faster than the grocery store test (192 photos).

9. Conclusion & Recommendations

Key Findings

  1. All 3 targets are directly detectable by GroundingDINO — unlike the grocery test where zero targets were found. Refrigerator (84.9%) and fire extinguisher (82.2%) have HIGH confidence.
  2. The office environment is significantly easier for GroundingDINO than the grocery store (avg 52% vs 41.2% confidence). Objects are distinct and well-separated.
  3. Left vs right door is the hardest challenge — GroundingDINO cannot determine spatial direction. The VLM must handle this through scene reasoning or user questioning.
  4. Test 2 (blind) is more efficient for the office due to the small space (6 min vs 14 min), but Test 1 (full photo) provides higher accuracy (85-90% vs 75-85%).
  5. Fire extinguisher is visible from the START position (IMG_1434 at 0.69), which may cause premature ARRIVED if not handled by sequential goal logic.
  6. No systematic false positives like the grocery store's "bathroom restroom" issue. Cabinet overdetection (62x) is the only noise, and it's generally correct.
