7. Error Analysis

Errors & Issues Encountered

ERROR 1: Systematic Misclassification — "bathroom restroom" (CRITICAL)

GroundingDINO detects "bathroom restroom" in ALL 192 photos at an average confidence of 50.3%. This is clearly a false positive — the photos are from a grocery store, not a bathroom. The tiled/reflective flooring and white lighting typical of supermarkets triggers the bathroom detector.

Impact: Wastes 1 of the 5 detection slots per photo. Reduces useful detection capacity by 20%.

Mitigation: Add "bathroom" and "restroom" to a negative prompt list, or post-filter detections to exclude labels that appear in >90% of photos.

ERROR 2: Missing Grocery-Specific Objects (HIGH)

None of these navigation-critical objects were detected:

  • cheese — 0 detections
  • frozen dumplings — 0 detections
  • checkout / cashier / cash register — 0 detections
  • shopping cart / basket — 0 detections
  • price tag / barcode — 0 detections
  • shelf / aisle marker — 0 detections
  • freezer / glass door — 0 detections (only generic "refrigerator")

Root Cause: The GroundingDINO prompt used generic object classes (the "find everything" goal) rather than grocery-specific terms. The detection prompt needs to include: "cheese . dairy . milk . yogurt . freezer . frozen food . checkout . cashier . cash register . shopping cart . aisle sign . price tag ."

Impact: System cannot directly locate any of the 3 navigation targets using object detection alone.

ERROR 3: Low Average Confidence (MODERATE)

Overall average confidence is 41.2%, with 86% of detections in the 30-45% range. This is near the box_threshold of 0.30, meaning the model is operating at the edge of its detection capability.

Root Cause: Supermarket environments are visually complex with repetitive patterns (shelves, products, labels). GroundingDINO's SwinT backbone may not have sufficient resolution for fine-grained grocery item recognition.

Impact: Higher false positive rate. Navigation decisions based on low-confidence detections may be unreliable.

ERROR 4: Ambiguous Label Merging (LOW)

Several detections have merged/confused labels:

  • cabinet refrigerator (4 detections) — model uncertain between cabinet and refrigerator
  • door refrigerator elevator (2 detections) — three labels merged
  • printer water dispenser (2 detections) — confused between two appliances
  • coffee machine water dispenser (1 detection) — similar form factors

Impact: Reduces precision of spatial reasoning. The topological map may assign wrong landmark labels to zones.

ERROR 5: VLM Warm-up Latency (OPERATIONAL)

First photo upload takes 30-60 seconds while both GroundingDINO and LLaMA 3.2 Vision load into VRAM. Subsequent calls are faster (~10-15s per photo).

Impact: For Test 1 (192 photos), total processing time estimated at: 60s + (191 x 12s) = ~40 minutes. For Test 2, each step takes ~15-30s.

ERROR 6: No SAM Segmentation (DESIGN LIMITATION)

SAM (Segment Anything Model) is intentionally disabled to fit within 8GB VRAM budget. Only GroundingDINO bounding boxes are available — no pixel-level masks.

Impact: Cannot precisely outline objects or measure their relative positions. Navigation relies more heavily on VLM scene understanding.

9. Conclusion & Recommendations

Key Findings

  1. GroundingDINO is not sufficient alone for grocery navigation. The generic prompt classes miss all grocery-specific items (cheese, dumplings, checkout). The VLM is the actual workhorse for target identification.
  2. Systematic "bathroom restroom" false positives waste 26% of all detections (207 out of 788). This must be filtered.
  3. "Refrigerator" is the best proxy for finding dairy and frozen sections (62 detections, best confidence 67.5%), but cannot distinguish between dairy fridges and freezers.
  4. Test 2 (blind navigation) is more practical for real-world use despite lower accuracy, because it doesn't require pre-mapping the entire store.
  5. The supermarket floor plan (18 aisles) built from the 192 photos provides accurate layout data, but the navigation system needs grocery-specific detection prompts to be effective.
